# .dev/ — Development artefacts

Retained for transparency. Not part of the published software; not built,
not tested, not part of the CPC submission.

## Contents

- `prompts/` — LLM workflow inputs (prompt files passed to AI assistants during development)
- `reports/` — LLM-generated optimisation and progress reports
- `a2dmrg-history/` — A2DMRG session logs and status files from incremental AI-assisted development

These files document how the codebase was developed. Reviewers may inspect them
for transparency but should not expect them to be functional or up to date.
