# pdmrg-gpu-base (reference baseline, not published)

Pre-optimisation snapshot of the corresponding published variant, preserved for diff reference.

This directory is retained so future contributors can compare the before/after of optimisation
passes. No results from this variant appear in the paper and it is not part of the CPC submission:

- It is not built by `gpu-rocm/build_all.sh`.
- It is not covered by the reproducibility checklist in `REPRODUCIBILITY.md`.
