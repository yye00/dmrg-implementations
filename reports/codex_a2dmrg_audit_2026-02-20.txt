4) ENV CACHING BYPASS / FAILURE MODES

- Two-site “cached” path still calls uncached microstep:
  - `a2dmrg/a2dmrg/parallel/local_steps.py:173` -> `local_microstep_2site(...)`, which rebuilds environments internally (`a2dmrg/a2dmrg/numerics/local_microstep.py:346`).
- Silent cache miss/fallback:
  - If env bond dims mismatch, one-site update is skipped and returns unchanged state + zero energy:
    - `a2dmrg/a2dmrg/numerics/local_microstep.py:87`, `a2dmrg/a2dmrg/numerics/local_microstep.py:221`.
  - This can hide invalid cache compatibility and degrade convergence without explicit signal.
- Cache construction itself duplicates/canonizes MPS twice per sweep:
  - `a2dmrg/a2dmrg/environments/cached_environment.py:80`, `:87`.

5) COARSE-SPACE O(K^2) / O(L^2) TRAPS

K = number of candidates (up to L+1 before filtering)
- Energy prefilter computes energy for every candidate, then allgathers:
  - `a2dmrg/a2dmrg/parallel/coarse_space.py:181`, `:189`, `:196`.
- Overlap redundancy filter is nested pairwise overlap checks:
  - `a2dmrg/a2dmrg/parallel/coarse_space.py:93` + norm overlaps at `:97`, `:98`.
- Coarse matrix assembly is quadratic in candidates (upper triangle pairs):
  - `a2dmrg/a2dmrg/parallel/coarse_space.py:250`, `:255`.
- Off-diagonal contractions build TN contraction per pair:
  - `a2dmrg/a2dmrg/parallel/coarse_space.py:294`, `:297`.
- Allreduce combines full coarse matrices:
  - `a2dmrg/a2dmrg/parallel/coarse_space.py:320`, `:321`.

Mitigation already present but limiting
- Hard cap `max_candidates=8` in coarse builder:
  - `a2dmrg/a2dmrg/parallel/coarse_space.py:126`.
- If cap is raised or disabled, quadratic behavior will reappear quickly.

6) CONVERGENCE / TOLERANCE MISMATCH VS QUIMB

Observed mismatches
- Mixed tolerance policy:
  - Main tol user-driven (`a2dmrg/a2dmrg/dmrg.py:494`), but compression cutoff fixed `1e-12` (`a2dmrg/a2dmrg/dmrg.py:467`, `:514`), and coarse regularization fixed `1e-8` (`a2dmrg/a2dmrg/dmrg.py:426`).
  - Warmup/finalization use quimb DMRG with fixed tight tolerances (`a2dmrg/a2dmrg/dmrg.py:289`, `:548`).
- Test/expectation mismatch:
  - README frames paper tolerance around 1e-6 (`a2dmrg/README.md:189`), but one test enforces 1e-10 exact match (`a2dmrg/a2dmrg/tests/test_l20_validation.py:45`).
  - Separate practical test explicitly relaxes tol to 1e-4 due runtime/convergence reality (`a2dmrg/a2dmrg/tests/test_l20_practical.py:36`).
- Net effect: benchmark/runtime comparisons vs quimb can be apples-to-oranges unless warmup/finalize/compression/regularization are aligned for both methods.
